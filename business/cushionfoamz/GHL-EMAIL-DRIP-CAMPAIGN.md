## ⚙️ GHL WORKFLOW SETUP

### Step 1: Create Custom Fields

**In GHL Contacts:**
1. Go to Settings → Custom Fields
2. Create these fields:

| Field Name | Type | Purpose |
|------------|------|---------|
| `cushion_type` | Text | Sofa, Window Seat, Outdoor, etc. |
| `quote_amount` | Number | Dollar amount of quote |
| `quote_date` | Date | When quote was sent |
| `quote_expiry` | Date | Quote expiration date |
| `fabric_preference` | Text | Sunbrella, Crypton, etc. |
| `project_timeline` | Text | Rush, Standard, Flexible |
| `quote_status` | Text | Pending, Approved, Expired, Won |

### Step 2: Create Tags

**Tags for This Campaign:**
- `quote-requested`
- `quote-sent`
- `quote-expired`
- `quote-won`
- `quote-lost`
- `email-drip-active`

### Step 3: Build the Workflow

**Workflow Name:** "Quote Request Nurture Sequence"

**Trigger:** Form Submitted → "Quote Request Form"

**Workflow Steps:**

1. **Action:** Add Tag `quote-requested`
2. **Action:** Add Tag `email-drip-active`
3. **Action:** Send Email 1 (Immediate Confirmation)
4. **Wait:** 2 days
5. **Action:** Update Custom Field `quote_date` = Current Date
6. **Action:** Send Email 2 (Quote Delivery)
7. **Action:** Add Tag `quote-sent`
8. **Wait:** 5 days
9. **Action:** Send Email 3 (Educational)
10. **Wait:** 5 days
11. **Action:** Send Email 4 (Reminder)
12. **Wait:** 3 days
13. **Action:** Send Email 5 (Last Chance + Bonus)
14. **Wait:** 1 day
15. **Condition:** Check if `quote_status` = "Approved"
   - **YES:** Remove Tag `email-drip-active`, Exit Workflow
   - **NO:** Continue
16. **Action:** Update Custom Field `quote_status` = "Expired"
17. **Action:** Add Tag `quote-expired`
18. **Action:** Remove Tag `email-drip-active`
19. **Wait:** 7 days
20. **Action:** Send Email 6 (Win-Back)
21. **Wait:** 30 days
22. **Action:** Send Email 7 (Long-Term Nurture)
23. **Action:** Remove Tag `quote-sent`

### Step 4: Create Quote Form

**Form Fields:**
- First Name (required)
- Last Name (required)
- Email (required)
- Phone (required)
- Cushion Type (dropdown: Sofa, Window Seat, Outdoor, Boat, RV, Dining, Bench, Other)
- Dimensions (text area)
- Fabric Preference (dropdown: No preference, Sunbrella, Crypton, Cotton, Other)
- Timeline (dropdown: Rush (1 week), Standard (2-3 weeks), Flexible)
- Budget Range (dropdown: Under $500, $500-1000, $1000-2500, $2500+)
- Additional Details (text area)
- How did you hear about us? (dropdown)

**Form Settings:**
- Submit Action: Redirect to Thank You page
- Thank You Page Message: "Thanks! We'll send your quote within 48 hours."
- Notification: Email [your-email] on submission
