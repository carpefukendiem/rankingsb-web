## 🚀 HOW TO IMPORT (2 Minutes)

### Step 1: Download the File
The file `ruben-calendar-import.ics` is in your workspace root.

### Step 2: Import to Google Calendar

**Option A: Web Browser (Easiest)**
1. Go to https://calendar.google.com
2. Click **+** next to "Other calendars" (left sidebar)
3. Select **"Import"**
4. Click **"Select file from your computer"**
5. Choose `ruben-calendar-import.ics`
6. Select calendar: **rubenstips@gmail.com**
7. Click **"Import"**

**Option B: Google Calendar Settings**
1. Go to https://calendar.google.com/calendar/u/0/r/settings/export
2. Under "Import" section, click **"Select file from your computer"**
3. Choose `ruben-calendar-import.ics`
4. Select your calendar
5. Click **"Import"**
