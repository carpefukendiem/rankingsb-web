## GHL AUTOMATION SETUP

### How to Build These in GoHighLevel

1. **Dashboard → Marketing → Campaigns**
2. **Create New Campaign**
3. **Add Workflow**
4. **Set Trigger:** Tag added, form submitted, etc.
5. **Add Email Steps:**
   - Delay: 0 minutes (immediate)
   - Delay: 1 day
   - Delay: 2 days
   - etc.
6. **Add If/Else Logic:**
   - If appointment booked → exit sequence
   - If replied → notify Sal
   - If not opened → send SMS instead

### Smart Tags to Use

- `Lead - New` → Triggers nurture sequence
- `Proposal Sent` → Triggers proposal follow-up
- `No Show` → Triggers re-engagement
- `Customer - Active` → Triggers onboarding
- `Customer - 60 Days` → Triggers referral ask

### SMS Fallback

For emails not opened within 48 hours, send SMS:
```
Hi [Name], Sal from Rankingsb. Emailed you about the SEO audit. Quick question - are you still interested in ranking higher? Reply YES and I'll call you. - Sal
```

## CUSTOMIZATION GUIDE

### Personalize These Variables:
- `{{contact.first_name}}` - First name
- `{{contact.company_name}}` - Business name
- `{{industry}}` - Their industry (HVAC, plumbing, etc.)
- `[specific findings]` - From their actual audit
- `[CALENDAR LINK]` - Your Calendly or GHL booking link

### Tone Adjustments:
- **Formal industries** (legal, medical): Use full sentences, proper grammar
- **Casual industries** (home services): Use contractions, conversational
- **High-end clients** (Montecito): Emphasize exclusivity, white-glove service

| Metric | Good | Great |
