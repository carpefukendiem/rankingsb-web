# Productized Offers (easy to sell, easy to deliver)

## 1) Lead Flow Engine (GHL) — core offer
**$999 setup + $399/mo (3-month minimum)**

### Who it’s for
Local service businesses that win by answering first and following up fast (electrician, pest control, HVAC, etc.).

### What’s included (setup)
- Pipeline stages + ownership
- Missed-call text-back
- New lead instant SMS + follow-up sequence
- Appointment reminders
- Review request sequence
- Tracking basics + handoff

### What’s included (monthly)
- Hosting/maintenance of automations
- Light tuning (small changes)
- Monthly performance snapshot

### Boundaries (to keep it SaaS-like)
- Not a “we run all your marketing” retainer
- No unlimited web/SEO/creative inside $399/mo

## 2) Google Ads “Lite Ops” — add-on or standalone
**$750 setup + $299/mo** (+ client ad spend paid by client)

### What’s included
- Account audit + conversion tracking check
- Campaign structure + keywords/negatives
- Basic landing page alignment recommendations
- Weekly optimization cadence (lightweight)
- Monthly report summary

### Boundaries
- Limited number of campaigns (define per client)
- Landing page builds not included (can be separate)

---

## 3) Website + Local SEO Fix Pack (non-GHL sites)
**$999 setup + $199/mo**

### What’s included (setup)
- On-page fixes for core pages (titles/H1s, internal links, speed basics)
- Local SEO essentials (GBP cleanup checklist + review plan)
- 1 local landing page outline + implementation notes

### What’s included (monthly)
- 1–2 small site/GBP optimizations
- Monthly check-in + KPI snapshot

---

## Simple sales rule
Start with **Lead Flow Engine**.
If they already have marketing but leak leads → Lead Flow Engine.
If they want more volume immediately and have budget → add Google Ads Lite.
If the website/SEO foundation is broken → Website + Local SEO Fix Pack.
