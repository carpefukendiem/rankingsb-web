## System Overview

A repeatable lead gen pipeline that produces qualified local-business leads and converts them into booked audit calls.

**Current Channel:** Outside sales (cold calling)  
**Target:** Local service businesses (starting with electricians)  
**Conversion Model:** Free audit → value demonstration → close

- **Setup:** $1,995 one-time
- **Monthly:** $297/month
- **Commission:** $997 per close (50% to Sal)

### Audit-First Sales Process
1. Cold call: "Free 10-minute SEO audit?"
2. Run 10-minute audit (5-6 key metrics)
3. Create 1-page PDF report
4. 15-minute review call
5. Pitch service
6. Close or nurture
