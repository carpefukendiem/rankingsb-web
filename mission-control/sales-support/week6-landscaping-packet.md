## 🔍 LANDSCAPING AUDIT SNIPPETS

### Snippet 1: GBP Category Check
```
❌ ISSUE: Primary category is "Landscaper" — missing "Lawn Care Service" 
and "Garden Center" which get more searches

✅ FIX: Add all relevant categories. "Lawn Care Service" gets 3x more 
searches than "Landscaper" in Santa Barbara.

💰 IMPACT: +15-20 calls/month from better categorization
```

### Snippet 2: Photo Strategy
```
❌ ISSUE: Only 8 photos on GBP. Competitors have 50+.

✅ FIX: Upload 5 before/after photos weekly. Add geo-tags to images 
(Santa Barbara locations). Name files with keywords: 
"santa-barbara-front-yard-landscaping.jpg"

💰 IMPACT: Photos increase clicks by 42%. More photos = more visibility.
```

### Snippet 3: Seasonal Keyword Gap
```
❌ ISSUE: Not ranking for "spring cleanup Santa Barbara" or 
"lawn aeration Goleta" — high-intent seasonal searches

✅ FIX: Create service pages for each seasonal offering. Spring 
services should be live by March 1st to capture early bookings.

💰 IMPACT: Seasonal keywords convert at 8% vs 2% for generic terms.
```

### Snippet 4: Review Velocity
```
❌ ISSUE: 12 reviews, last one 3 months ago. Competitors add 5+/month.

✅ FIX: Set up automatic review requests after job completion. 
Target: 10 new reviews this month. Reply to ALL reviews within 24hrs.

💰 IMPACT: Review velocity is a ranking factor. Fresh reviews = fresh signals.
```

### Snippet 5: Citation Inconsistency
```
❌ ISSUE: NAP (Name/Address/Phone) inconsistent across directories:
- Yelp: "Santa Barbara Lawn Care"
- Angi: "SB Lawn & Landscape"  
- BBB: Different phone number

✅ FIX: Standardize NAP everywhere. Even small variations hurt rankings.

💰 IMPACT: Consistent citations can boost rankings 10-15 positions.
```

### Snippet 6: Website Mobile Issues
```
❌ ISSUE: Website takes 6 seconds to load on mobile. 53% of mobile 
users leave after 3 seconds.

✅ FIX: Compress images, enable caching, minify code. Target: <3 seconds.

💰 IMPACT: Each 1-second delay = 7% conversion loss. Speed matters.
```

### Snippet 7: Service Area Confusion
```
❌ ISSUE: GBP shows "Santa Barbara" but website says 
"Serving Santa Barbara County" — Google can't confirm service area.

✅ FIX: Match service area across all platforms. List specific cities: 
Santa Barbara, Goleta, Carpinteria, Montecito.

💰 IMPACT: Clear service area = better local pack rankings.
```

### Snippet 8: No Blog/Content
```
❌ ISSUE: Zero blog posts. Missing "when to fertilize lawn in Santa Barbara" 
and other informational searches.

✅ FIX: Publish 2 posts/month: "Santa Barbara Spring Lawn Care Guide" 
and "Best Drought-Tolerant Plants for Santa Barbara Gardens"

💰 IMPACT: Content drives 3x more organic traffic than service pages alone.
```
